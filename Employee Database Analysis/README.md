## Employee database analysis

SQL used to combine multiple csvs, python in a jupyter notebook used to explore the data.

Original data can be found in the "data" folder

### Data structuring, done in sql, can be veiwied in the employee_database.sql file. This is not meant to be run all at once, and the data must be manually imported into each table.

### A database diagram export can be found as QuickDBD-export.sql, and is best previewed on https://app.quickdatabasediagrams.com/#/

### Finally, the analysis can be found as "Employee Databse Analysis.ipynb"
